HTML et CSS

L'HyperText Markup Language, HTML, désigne un type de langage informatique descriptif. Il s'agit d'un format de données utilisé dans l'univers d'Internet pour la mise en forme des pages Web.
CSS est l'acronyme de « Cascading Style Sheets » ce qui signifie « feuille de style en cascade ».Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML).

Les points principaux dans cette formation sont :

- Gérer l’importance de texte en HTML.
- Listes ordonnées, non ordonnées, descriptions et imbriquées.
- Créer des liens interne et externe.
- Insérer des photos, des audios et des vidéos.
- Création des tableaux.
- Intégrer CSS avec HTML et gérer les styles d’une page.
- Police d’écriture en CSS et gestion de couleur.
